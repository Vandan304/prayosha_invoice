DEPLOYMENT GUIDE: PRAYOSHA INVOICE

1. GITHUB PUSH
- Make sure you push the latest version with the new "vercel.json".

2. VERCEL SETUP (TO AVOID 404)
- Import repo in Vercel Dashboard.
- IMPORTANT: Leave "Root Directory" empty (project root).
- Framework: Vite.

3. ENVIRONMENT VARIABLES (REQUIRED)
In Vercel Settings > Environment Variables, add:
- MONGO_URI: (Your MongoDB Atlas connection URL)
- NODE_ENV: production
- VITE_API_URL: /api

4. TROUBLESHOOTING
- If "/" gives 404, check if "vercel.json" is in the root of your GitHub repo.
- If "/api" gives 404, check "Functions" tab in Vercel.

VERIFICATION
- Site: https://your-project.vercel.app/
- API: https://your-project.vercel.app/api
